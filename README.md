### Controle para jogar `Guitar Hero` no PC utilizando `ARDUINO PRO MICRO / LEONARDO`

GPIO da placa: [Arduino Pro Micro GPIO](https://cdn.discordapp.com/attachments/663479366918995985/770460915350765648/pro_micro_pinout.png)

Faça download da [IDE do Arduino](https://www.arduino.cc/en/Main/Software#)

Instale a biblioteca [Joystick.h](https://github.com/MHeironimus/ArduinoJoystickLibrary)

## Abra o arquivo: `guitar_hero_controller.ino`
- `FERRAMENTAS > Placa > Arduino Micro`
- `FERRAMENTAS > Programador > Arduino as ISP`
- `Compile o código e suba para o arduino.`
